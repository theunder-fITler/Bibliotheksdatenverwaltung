# Bibliotheksdatenverwaltung

> Ein Übungsprojekt zur Datenverwaltung und Digitalisierung

## Projektbeschreibung

Dieses Projekt entstand im Rahmen meiner Umschulung zum Fachinformatiker Daten- und Prozessanalyse.

**Die Idee:**  
Im Lehrbuch fand ich folgende Aufgabe:
```
Führen Sie für eine selbstgewählte Aufgabe aus dem Bereich Softwareentwicklung mithilfe eines IPERKA-Plans (siehe Anhang) eine Anforderungsspezifikation durch und erstellen Sie danach ein Lasten- und ein Pflichtenheft. 

Die Aufgabe sollte als Kernziel eine Verwaltung von Daten beinhalten. Dabei können das Thema und die Daten, welche verwaltet werden, frei bestimmt werden. Diese können aus dem beruflichen oder dem privaten Umfeld kommen. 
```

**Das Ziel:**
Verknüpfung der Umschulungsinhalte zu einem praxisnahem Projekt:
- also eine Datenbank in guter Form erstellen,
- Schnittstelle zu py herstellen,
- Datenexploration mit pandas,
- Datenbereinigung mit py,
- selbstständiges Arbeiten in einer praxisnahen Form,
- Skills und Theorie aus dem PM einbinden
